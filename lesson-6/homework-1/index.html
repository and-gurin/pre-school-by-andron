Значения тегов HTML

dl
Список описаний, наименований
dt
Название термина
dd
Определение, описание термина
abbr
Аббревиатура (сокращение наименования до заглавных символов)
address
Для контактной информации
aside
Для боковой панели документа
code
Для отображения участка кода
caption
Заголовок для таблицы
cite
Название книги, или фильма
fieldset
Группирует связанные поля в форме
figcaption
Заголовок, или подпись для тега figure
legend
Заголовок для группы элементов формы
mark
Выделяет часть текста
output
Для вывода результата расчета
optgroup
Группирует связанные элементы в теге select
option
Определение элемента из списка опций
picture
Позволяет выводить изображения в разных разрешениях
progress
Отображает ход выполнения задачи
section
Раздел в документе
samp
Для вывода фрагмента кода моноширинным шрифтом
summary
Заголовок для элемента details
tfoot
Нижняя часть таблицы
ер
Ячейка заголовка таблицы
thead
Верхняя часть таблицы
time
Для отображения времени или даты
track
Указывает текстовые дорожки в элементах audio и video
meter
Отображает величину в пределах определенного диапазона
nav
Блок навигации
menu
Отображает список команд
menuitem
Отображает команду которую можно вызвать из контекстного, или другого меню
keygen
Используется для генерации пары ключей для форм
main
Главная часть контента
embed
Контейнер для отображения медиа объектов
Можно ли дважды на странице использовать h1?
Нет
Можно ли дважды на странице использовать main?
Нет
Можно ли дважды на странице использовать article?
Да
Чем отличается b от strong?
B просто выделяет текст, strong выделяет визуально и по смыслу
Чем отличается URI от URL?
URI -- идентификатор ресурса. URL -- адрес ресурса
